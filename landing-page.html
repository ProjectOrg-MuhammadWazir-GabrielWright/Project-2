What is sentiment analysis?

A way to determine how a user feels based on the language that they use by combining natural language processing and text analysis

How does it work?

The tool:

  1. Breaks down each comment into smaller sentences
  2. Identifies the relationship between each sentence
  3. Evaluates the above to find the overall sentiment

Sentiment is expressed as:

  • Positive
  • Negative

The tool also factors in:

  • Irony
  • Subjectivity
